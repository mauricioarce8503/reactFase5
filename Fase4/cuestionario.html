Cuestionario de CSS Grid

1. ¿Qué es CSS Grid?

a) Una técnica para diseñar layouts con CSS.
b) Un sistema de cuadrícula para dibujar en el navegador.
c) Una librería de JavaScript para crear gráficos en el navegador.

2. ¿Cuáles son los dos componentes principales de CSS Grid?

a) Filas y columnas.
b) Cajas y contenedores.
c) Líneas de cuadrícula y celdas.

3. ¿Cómo se define una cuadrícula en CSS Grid?

a) Con la propiedad grid-template.
b) Con la propiedad grid-area.
c) Con la propiedad grid-row.

4. ¿Cuál es la sintaxis para seleccionar un área en CSS Grid?

a) grid-row: start / end; grid-column: start / end;
b) grid-template-areas: "nombre-del-area";
c) grid-area: nombre-del-area;

5. ¿Cómo se define el espacio entre filas y columnas en CSS Grid?

a) Con la propiedad grid-gap.
b) Con la propiedad grid-template-rows y grid-template-columns.
c) Con la propiedad justify-content y align-content.
